== Description ==
 
Serengeti builders post grid offers limitless and full control of how you want your 
grids to appear, such as background colors, styles and so on.
also you can easily specify how you want your grids to open ( on default 
post url, or custom url) 
 
 
== Installation ==
  
1. Upload/Install plugin via wp plugin dashboard
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Serengeti grid will will appear on left wp menu, click on setting
4. Add new grid
5. go to grid list and customize your grid (how should appear & which post category to display)
